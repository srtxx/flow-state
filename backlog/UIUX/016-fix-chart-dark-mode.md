# チャート背景色のダークモード対応

**優先度**: 🟡 中  
**カテゴリ**: Bugfix / UI  
**見積もり時間**: 15分

## 問題
アプリ全体をダークモードにしても、AlertnessChartの背景色が白（#fff）のまま固定されており、視覚的に浮いている。

## 原因
`AlertnessChart.tsx` のインラインスタイルで `backgroundColor: '#fff'` がハードコードされている。

## 解決策
CSS変数 `var(--bg-card)` を使用して、テーマに応じて背景色が切り替わるように修正する。

## 対象ファイル
- `src/components/AlertnessChart.tsx`

## 完了条件
- [ ] ダークモード時にチャート背景が暗色（#1a1a1a 等）になること。
- [ ] ライトモード時は引き続き白背景であること。
