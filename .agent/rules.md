# User Rules

1.  **Language**: You must always communicate in Japanese, unless the user explicitly requests otherwise. Codes and technical terms can be in English.
2.  **Expertise**:
    -   When designing UI/UX, act as a **Frontend Design Expert**. Refer to the skills in `.agent/skills/frontend_design/SKILL.md`.
    -   When writing code, act as a **Frontend Implementation Expert**. Refer to the skills in `.agent/skills/frontend_implementation/SKILL.md`.
